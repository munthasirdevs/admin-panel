# Toast

A flexible toast notification system with multiple variants, positions, and auto-dismiss functionality.

## Installation

```tsx
import { ToastProvider, useToast } from '@components';
```

## Setup

Wrap your application with `ToastProvider`:

```tsx
import { ToastProvider } from '@components';

function App() {
  return (
    <ToastProvider position="top-right">
      <YourApp />
    </ToastProvider>
  );
}
```

## Basic Usage

```tsx
import { useToast } from '@components';

function MyComponent() {
  const { success, error, info, warning } = useToast();

  const handleSave = () => {
    success('Changes saved successfully!');
  };

  const handleError = () => {
    error('Failed to save changes');
  };

  return (
    <>
      <Button onClick={handleSave}>Save</Button>
      <Button onClick={handleError}>Error</Button>
    </>
  );
}
```

## Options

### Custom Duration

```tsx
const { success } = useToast();

success('This will disappear in 10 seconds', {
  duration: 10000,
});
```

### Disable Auto-dismiss

```tsx
const { info } = useToast();

info('This will stay until dismissed', {
  duration: 0,
});
```

### Non-dismissible

```tsx
const { addToast } = useToast();

addToast({
  type: 'info',
  message: 'Processing...',
  dismissible: false,
});
```

### Custom ID

```tsx
const { addToast, dismissToast } = useToast();

const id = addToast({
  id: 'my-custom-id',
  type: 'info',
  message: 'Loading...',
});

// Later, dismiss by ID
dismissToast('my-custom-id');
```

## Positions

Configure position in `ToastProvider`:

```tsx
<ToastProvider position="top-right">
  <App />
</ToastProvider>
```

Available positions:
- `top-right` (default)
- `top-left`
- `top-center`
- `bottom-right`
- `bottom-left`
- `bottom-center`

## Methods

### addToast

Add a custom toast notification.

```tsx
const { addToast } = useToast();

const id = addToast({
  type: 'info',
  title: 'Title',
  message: 'Message',
  duration: 5000,
  dismissible: true,
});
```

### dismissToast

Dismiss a specific toast by ID.

```tsx
const { dismissToast } = useToast();
dismissToast('toast-id');
```

### dismissAll

Dismiss all toasts.

```tsx
const { dismissAll } = useToast();
dismissAll();
```

### success

Show a success toast.

```tsx
const { success } = useToast();
success('Message', options);
```

### error

Show an error toast.

```tsx
const { error } = useToast();
error('Message', options);
```

### warning

Show a warning toast.

```tsx
const { warning } = useToast();
warning('Message', options);
```

### info

Show an info toast.

```tsx
const { info } = useToast();
info('Message', options);
```

## Props

### ToastProvider Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultDuration` | `number` | `5000` | Default duration in ms |
| `maxToasts` | `number` | `5` | Maximum visible toasts |
| `position` | `ToastPosition` | `'top-right'` | Toast container position |
| `className` | `string` | - | Additional CSS classes |

### Toast Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `type` | `'success' \| 'error' \| 'warning' \| 'info'` | - | Toast type |
| `title` | `string` | - | Toast title |
| `message` | `string` | - | Toast message |
| `duration` | `number` | `5000` | Auto-dismiss duration (0 to disable) |
| `dismissible` | `boolean` | `true` | Allow manual dismiss |
| `id` | `string` | auto-generated | Custom toast ID |

## Accessibility

- Toasts have `role="alert"` for screen readers
- `aria-live="assertive"` for immediate announcements
- Dismiss buttons have proper `aria-label`
- Focus is not trapped (toasts are non-blocking)

## Best Practices

1. **Use appropriate types**: Success for positive outcomes, error for failures
2. **Keep messages brief**: Toasts should be quickly readable
3. **Don't overuse**: Too many toasts can be overwhelming
4. **Provide actions**: For important toasts, consider adding actions
5. **Respect duration**: Use longer durations for important information

## Examples

### Form Submission

```tsx
function ContactForm() {
  const { success, error } = useToast();
  const [isSubmitting, setIsSubmitting] = useState(false);

  const handleSubmit = async (data) => {
    setIsSubmitting(true);
    try {
      await submitForm(data);
      success('Message sent successfully!');
    } catch (err) {
      error('Failed to send message. Please try again.');
    } finally {
      setIsSubmitting(false);
    }
  };

  return <form onSubmit={handleSubmit}>...</form>;
}
```

### Delete Confirmation

```tsx
function DeleteButton({ id }) {
  const { success, error } = useToast();

  const handleDelete = async () => {
    try {
      await deleteItem(id);
      success('Item deleted successfully');
    } catch (err) {
      error('Failed to delete item');
    }
  };

  return <Button variant="danger" onClick={handleDelete}>Delete</Button>;
}
```

### Upload Progress

```tsx
function UploadButton() {
  const { addToast, dismissToast } = useToast();

  const handleUpload = async () => {
    const id = addToast({
      type: 'info',
      title: 'Uploading',
      message: 'Please wait...',
      dismissible: false,
      duration: 0,
    });

    try {
      await uploadFile();
      dismissToast(id);
      addToast({
        type: 'success',
        title: 'Upload Complete',
        message: 'Your file has been uploaded',
      });
    } catch (err) {
      dismissToast(id);
      addToast({
        type: 'error',
        title: 'Upload Failed',
        message: 'Please try again',
      });
    }
  };

  return <Button onClick={handleUpload}>Upload</Button>;
}
```

### Multiple Toasts

```tsx
function BatchOperation() {
  const { success, error, warning } = useToast();

  const handleBatch = async () => {
    const results = await processBatch();
    
    if (results.success > 0) {
      success(`${results.success} items processed successfully`);
    }
    if (results.failed > 0) {
      error(`${results.failed} items failed to process`);
    }
    if (results.skipped > 0) {
      warning(`${results.skipped} items were skipped`);
    }
  };

  return <Button onClick={handleBatch}>Process Batch</Button>;
}
```
